# Frontend assets

This folder contains the lightweight frontend assets used by the Gradio Space.

The app is still launched from the repository root with:

```bash
python app.py
```

`styles.css` holds the custom visual layer for Headline Booster: sidebar, topbar, welcome avatar, chat bubbles, responsive layout, and composer styling. Keeping it in `frontend/` makes the UI easier to edit without reintroducing the previous React/Vite stack.
